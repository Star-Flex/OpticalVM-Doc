代理商返利是原光云推出的推广奖励计划，允许代理商通过推广原光云产品获得返利收益。成为代理商后，您的下级用户每次新购云产品时，您都可以获得一定比例的返利奖励。

## 如何成为代理商

<Steps>

1. **访问返利推广页面**

   登录原光云控制台，进入 **返利推广** 页面。

2. **填写联系方式**

   按照页面提示填写您的联系方式（微信、QQ、邮箱等）。

   :::caution[注意事项]
   联系方式将展示给您的下级用户，方便他们联系您咨询和了解更多信息，请确保填写准确。
   :::

   ![rebate-page](@assets/rebate/1.webp)

3. **开始推广**

   注册成功后，您即可获得专属的推广链接和返利码，开始推广赚取返利。

</Steps>

## 返利流程

代理商返利的完整流程如下：

<Steps>

1. **绑定下级用户**

   将您的专属推广链接发送给已注册的用户，或者发送返利码让用户在 **账户中心 - 代理商信息** 中填写。

   ![bind-page](@assets/rebate/2.webp)

   绑定成功后，该用户将成为您的下级用户，您可以在 **下级管理** 页面查看所有下级用户列表。

2. **用户新购产品**

   当下级用户新购云产品（如云服务器 ECS）时，您会实时收到相应的返利金额。

   :::caution[注意事项]
   仅 **新购** 产品会产生返利，用户的 **续费、升级、降配** 等操作均不产生返利。
   :::

3. **返利进入冻结期**

   返利金额会立即进入 **30 天冻结期**。冻结期内，返利金额暂时无法提现。

4. **冻结期结束**

   冻结期结束后，返利金额会自动进入 **可提现** 状态，您可以申请提现到支付宝账户。

</Steps>

### 退款对返利的影响

如果下级用户在购买产品后发起退款：

- **冻结期内退款**：对应的返利金额将被直接扣除
- **冻结期后退款**：即使返利已变为可提现状态，对应的返利金额仍会被扣除
- **已提现后退款**：如果返利已经提现到您的账户，用户再退款时，您的可提现余额会被扣除相应金额，**可能导致余额变为负数**

:::danger[重要风险提示]

**无论返利是否已过冻结期、是否已提现，只要用户退款，相关的返利金额都会被系统自动扣除。**

**特别注意：** 如果您已经将返利提现，而用户在之后发起退款，您的返利账户余额可能会变成 **负数**。这种情况下，您需要通过后续的返利收入来弥补负数余额，才能继续提现。

**建议代理商：**
- 谨慎推广，选择信誉良好的下级用户
- 不要在冻结期刚结束时立即全部提现
- 保留一定的返利余额以应对可能的退款

:::

**退款扣除计算示例：**

假设您的返利账户状态如下：
- 已提现：¥500
- 当前可提现余额：¥100
- 冻结中：¥200

此时，某下级用户退款，涉及返利 ¥300：
- 扣除后可提现余额：¥100 - ¥300 = **-¥200**（负数）
- 您需要等待后续返利累计到 ¥200 以上，才能再次提现

## 返利比例

代理商的返利比例会在 **返利推广** 页面公开展示，具体比例由以下因素决定：

- **账号等级**：等级越高，返利比例可能越高
- **限时活动**：特定活动期间可能提供更高的返利比例
- **运营策略**：根据网站运营策略动态调整

返利比例 = 用户实际支付金额 × 返利比例百分比

## 推广工具

### 长链接缩短工具

为了方便分享推广链接，原光云提供了长链接缩短工具。您可以将较长的绑定链接或产品配置链接进行缩短，生成简洁的短链接。

![short-tools](@assets/rebate/3.webp)

**使用示例：**

- **原始链接**：`https://www.opticalvm.com/products/cloud-server?aff=123456&config=xxx`
- **缩短后**：`https://dwz.cn/JpN0A2qv`

:::tip[提示]
该工具仅支持原光云站内链接的缩短，不支持外部链接。即`opticalvm.com`
:::

### 全站返利排行榜

系统会统计所有代理商的返利金额，进行排名展示，方便您了解自己的推广效果和在全站的排名情况。

![rank](@assets/rebate/4.webp)

:::caution[数据说明]
排行榜数据仅供参考，实际展示的金额可能存在以下情况导致的延迟或偏差：

- 用户退款导致的返利扣除
- 数据缓存更新延迟
- 冻结期内的返利金额

如需查看实时详细数据，请访问 **返利订单流水** 表单。
:::

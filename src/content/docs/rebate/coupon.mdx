代理商优惠券是原光云为代理商提供的一种推广工具，允许代理商通过发放优惠券来吸引用户购买原光云产品。当代理商有了第一笔返利收入后，即可在控制台的 **代理市场** 页面购买已上架的优惠券，用于发放给下级用户。用户在购买产品时使用优惠券可享受相应的折扣优惠。

:::tip[推广建议]
建议代理商根据不同的推广场景和用户群体，选择合适的优惠券类型和力度，制定灵活的发放策略，以达到最佳的推广效果。
:::

## 如何购买优惠券

<Steps>

1. **访问代理市场**

   登录原光云控制台，进入 **代理市场** 页面，查看当前已上架的优惠券包。

   ![coupon-market](@assets/rebate/5.webp)

2. **选择优惠券包**

   点击准备购买的优惠券包，查看包含的优惠券详情，包括：

   - 优惠券类型（折扣券、满减券等）
   - 优惠金额或折扣力度
   - 有效期限
   - 使用条件
   - 优惠券数量

3. **确认购买**

   点击 **立即购买** 按钮，系统将从您的 **返利钱包** 中扣除相应金额。

   :::caution[扣款说明]
   如果优惠券有设置价格，购买时将从返利钱包中扣款。请确保返利钱包余额充足。
   :::

4. **获得优惠券**

   购买成功后，优惠券将立即到账您的账户，可以在 **优惠券管理** 中查看和发放给下级用户。

</Steps>

## 优惠券管理

### 优惠券使用规则

下级用户收到优惠券后，在购买原光云产品时可以使用：

- 优惠券有有效期限制，过期自动失效
- 不同优惠券可能有使用条件（如满额使用、指定产品使用等）
- 使用优惠券的订单，代理商仍可获得相应的返利奖励

:::danger[重要提示]

1. **购买的优惠券不支持退款**，请谨慎选择购买数量
2. 优惠券购买后即进入您的账户，请妥善管理和发放
3. 已发放的优惠券无法收回，请确认下级用户信息后再发放
4. 优惠券有效期到期后自动失效，无法延期

:::

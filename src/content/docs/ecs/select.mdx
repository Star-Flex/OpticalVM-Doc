在选择 ECS（弹性计算服务，或者称为 云服务器，VPS）机型时，您需要根据您的应用需求和预算来做出决定。ECS 提供了多种机型，每种机型都有不同的计算能力、内存大小和存储选项。

## 机型分类

### 游戏型（推荐）

专为游戏服务器优化，提供高性能的单核处理能力和低延迟网络。**90% 的用户都选择这个类型用于运行 Minecraft 服务器。**

**适用场景**：

- Minecraft 服务器（强烈推荐）
- 泰拉瑞亚、饥荒等多人游戏服务器
- CS:GO、方舟等对性能要求高的游戏
- 需要低延迟网络的实时游戏应用

**特点**：配备高主频 CPU，优先保证游戏运行的流畅度和响应速度，即使在多人在线时也能保持稳定。

### 通用型

提供均衡的 CPU、内存和网络资源配比，适合非游戏类应用场景。

**适用场景**：

- Web 应用服务器和网站托管
- 个人博客、论坛
- 开发测试环境
- 机器人程序（QQ 机器人、Discord Bot 等）
- 轻量级应用服务

**推荐选择**：如果你不是用来开游戏服务器，而是运行其他应用，通用型是合适的选择。

:::tip[如何选择]

- **开 Minecraft 或其他游戏服务器？** → 选择**游戏型**（强烈推荐）
- **搭建网站、运行机器人或其他应用？** → 选择**通用型**
  :::

## 网络配置

### IP 类型

ECS 的公网 IP 配置方式可以分为以下几种类型：

- **独立 IP（独享 IP）**：每个实例拥有独立的公网 IP 地址，所有端口（1-65535）都可以自由使用。适合需要开放多个端口、运行多个服务或需要特定 IP 地址的应用场景。例如：Web 服务器、邮件服务器、数据库服务器等。

- **共享 IP（NAT 端口转发）**：多个实例共享一个或多个公网 IP 地址，通过端口映射的方式访问。每个实例只能使用分配给它的特定端口，无法使用所有端口。适合只需要开放少量端口的应用场景。例如：游戏服务器（通常只需要 1-2 个端口）、单一用途的应用服务等。

:::note[如何选择]

- 如果你需要运行 Web 服务（80/443 端口）+ SSH（22 端口）+ 其他服务，建议选择**独立 IP**
- 如果你只是运行单一的游戏服务器或应用（如 Minecraft 服务器），**共享 IP** 通常更经济实惠
- 独立 IP 的价格通常比共享 IP 高，但提供更大的灵活性
  :::

### 网络线路

不同的网络线路决定了服务器的访问速度和稳定性，以下是常见的网络线路类型：

#### 基础线路

- **BGP 多线**：通过 BGP 协议连接多个网络运营商（电信+联通+移动），服务器只需一个 IP 地址，路由器自动选择最优访问路径。异网访问也能够获得较好的网络质量，当某条线路故障时可自动切换到其他线路。（如电信用户可以直接访问电信骨干，联通用户访问联通骨干，减少延迟和丢包）
- **多线**：服务器配置多个运营商的独立 IP 地址（电信 IP + 联通 IP + 移动 IP），通过 DNS 智能解析让不同运营商用户访问对应的 IP。相比 BGP 多线，普通多线成本更低，但需要占用更多 IP 资源，且线路故障时无法自动切换。
- **电信单线**：仅通过中国电信网络连接。电信用户访问快，但联通/移动用户访问可能较慢。适合目标用户主要是电信网络的场景。
- **联通单线**：仅通过中国联通网络连接。联通用户访问快，但电信/移动用户访问可能较慢。适合目标用户主要是联通网络的场景。
- **移动单线**：仅通过中国移动网络连接。移动用户访问快，但电信/联通用户访问可能较慢。适合目标用户主要是移动网络的场景。

:::note
单线适合对延迟不敏感的应用，或是 Web 访问需要前置 CDN 的场景。对于国内服务器，建议优先选择 BGP 多线以获得更好的用户体验。
:::

#### 国际优化线路

针对跨境访问优化的线路，提供更好的国际出口质量：

- **CN2 GT 线路**：CN2 Global Transit 的缩写，是中国电信 CN2 线路的中端产品。国内段可能经过 202.97 传统节点，仅在国际出口才走 59.43 CN2 节点。相比 163 骨干网有明显提升，但仍不如 GIA 线路稳定。适合对网络质量有一定要求但预算适中的用户。
- **CN2 GIA 线路**：CN2 Global Internet Access 的缩写，是中国电信 CN2 线路的高端产品。去程和回程全程走 59.43 高速节点（AS4809），拥有独立的回国链路，线路稳定、速度快、丢包率低。适合对网络质量要求较高的关键应用场景。
- **AS9929 线路**：中国联通的高端优化线路（联通 A 网/CU VIP），性能与 CN2 GIA 相当，提供优质的国际出口体验。适合联通用户或需要联通线路优化的应用。
- **CMI 线路**：中国移动国际有限公司（China Mobile International）的网络线路，提供移动用户的国际出口优化。相比移动的普通线路有更好的国际访问表现。

#### 国际专线

最高端的网络连接方式，不经过公网的点对点专用线路：

- **IPLC 专线**：国际私有租用线路（International Private Leased Circuit），点对点的物理专线连接，不经过公网，具有最高的稳定性和隐私性。
- **IEPL 专线**：国际以太网专线（International Ethernet Private Line），基于以太网技术的国际专线，相比 IPLC 更灵活，支持更高带宽。

两种专线都价格昂贵，但提供最佳的网络质量和安全性，适合对网络质量和安全性有极高要求的企业应用。

:::tip[网络质量对比]
**国际专线**：IPLC 专线 ≈ IEPL 专线

**国际优化线路**：CN2 GIA ≈ AS9929 > CN2 GT > CMI

**基础线路**：BGP 多线 > 多线（电信+联通+移动） > 单线（电信/联通/移动）

:::

## 防御保护

### 什么是 DDoS 防护？

**DDoS（分布式拒绝服务攻击）** 是一种恶意攻击方式，攻击者通过大量的虚假请求淹没服务器，导致服务器无法正常响应合法用户的请求，最终使服务器崩溃或无法访问。

**为什么需要 DDoS 防护？**

- **游戏服务器容易成为攻击目标**：恶意玩家可能会对竞争对手的服务器发起攻击，导致游戏无法运行
- **保护服务稳定性**：没有防护的服务器一旦遭受攻击，可能会直接宕机，影响所有用户
- **避免经济损失**：服务器宕机会导致用户流失和业务中断

### DDoS 防护如何工作？

DDoS 防护通过以下方式保护您的服务器：

1. **流量清洗**：在流量到达服务器之前，防护系统会识别并过滤掉恶意流量，只让正常流量通过
2. **黑洞路由**：当检测到超大规模攻击时，会临时将攻击流量导入"黑洞"，保护网络基础设施
3. **智能识别**：通过机器学习和规则引擎，区分正常访问和攻击流量

### 防护容量说明

原光云提供的防护容量标注，例如 **"150G 防护"**，是什么意思？

- **150G** 指的是防护系统能够承受的最大攻击流量为 **150 Gbps**（千兆比特每秒）
- 当攻击流量超过 150G 时，防护系统可能无法完全抵御，服务器可能会被暂时封禁（进入黑洞）
- **常见的防护容量**：
  - 10G - 30G：基础防护，适合小型个人服务器
  - 50G - 100G：中等防护，适合中型游戏服务器
  - 100G - 300G：高级防护，适合大型服务器或重要业务
  - 300G+：企业级防护，适合对稳定性要求极高的场景

### 独立 IP 与共享 IP 的防护区别

:::danger[重要提醒]
防护资源的分配方式与 IP 类型直接相关：

**独立 IP（独享防护）**：

- 防护资源完全独享，例如购买了 150G 防护，这 150G 完全属于你一个人
- 不受其他用户影响，即使同机房其他服务器被攻击，你的防护资源不会被占用
- 更稳定可靠，推荐给对稳定性要求高的用户

**共享 IP（共享防护）**：

- 多个用户共享同一个 IP 地址，防护资源也是**共享的**
- 例如标注 150G 防护，但这 150G 是所有使用该 IP 的用户共同分享的
- 如果其中一个用户遭受攻击，会消耗共享的防护容量，可能影响到其他用户
- 适合预算有限且攻击风险较低的场景
  :::

:::tip[如何选择防护等级]

1. **Minecraft 小型服务器（10 人以下）**：10G - 30G 防护足够
2. **Minecraft 中型服务器（10-50 人）**：建议 50G - 100G 防护
3. **大型游戏服务器或竞技服务器**：建议 100G 以上防护
4. **经常被攻击的服务器**：建议选择独立 IP + 高防护容量
   :::

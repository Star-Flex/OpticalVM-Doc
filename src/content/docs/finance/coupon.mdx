我的卡包是您管理和使用优惠券的专属空间。在这里，您可以查看已拥有的优惠券、使用优惠券购买云产品，以及将优惠券赠送给下级用户。优惠券可以帮助您在购买原光云产品时享受折扣优惠。

:::tip[温馨提示]

- 优惠券有有效期限制，过期后自动失效
- 使用优惠券时请注意使用条件（如满额、指定产品等）
- 优惠券不可兑换现金，不可退款

:::

## 优惠券获取渠道

原光云的优惠券可以通过以下渠道获取：

- **特殊活动赠送**：参与原光云举办的促销活动、节日活动等
- **代理商发放**：从您的上级代理商处获得优惠券
- **社区活动领取**：参与原光云社区活动、论坛互动等获得奖励

## 优惠券类型

<Tabs>
<TabItem label="满减券">

#### 什么是满减券

满减券是指在满足一定金额条件下，直接减免相应金额的优惠券。

**使用示例：**

```txt
优惠券：满 ¥100 减 ¥20
订单金额：¥150
实付金额：¥150 - ¥20 = ¥130
```

**注意事项：**

- 订单金额必须达到满减门槛才能使用
- 减免金额固定，不受订单金额影响
- 某些满减券可能限定特定产品使用

</TabItem>

<TabItem label="折扣券">

#### 什么是折扣券

折扣券是指在满足一定条件下，按比例折扣的优惠券。

**使用示例：**

```txt
优惠券：满 ¥100 享 8 折
订单金额：¥150
实付金额：¥150 × 0.8 = ¥120
```

**注意事项：**

- 订单金额必须达到使用门槛
- 折扣后金额根据订单实际金额计算
- 折扣券可能设有最高优惠上限

</TabItem>
</Tabs>

## 使用优惠券

在购买原光云产品时，系统会自动展示可用的优惠券。您可以选择合适的优惠券进行抵扣：

1. 在订单确认页面，查看 **可用优惠券** 列表
2. 选择要使用的优惠券
3. 确认优惠后的金额无误
4. 完成支付

:::caution[使用注意事项]

- 优惠券使用后不可撤销
- **订单支付成功后退款**：已使用的优惠券不会退还
- **订单未支付自动释放**：订单超时自动取消时，优惠券会自动退还到您的卡包
- 请在有效期内使用，过期优惠券无法使用

:::

## 赠送优惠券

某些优惠券支持赠送给您的下级用户使用。通过赠送优惠券，您可以帮助下级用户降低购买成本，提升推广效果。

:::tip[前置条件]
要赠送优惠券，您需要先成为代理商并拥有下级用户。如果您还不是代理商，请先查看 <a href="/rebate/rebate" target="_blank">代理商返利</a> 了解如何免费注册成为代理商。
:::

### 赠送流程

<Steps>

1. **成为代理商**

   前往原光云控制台的 **返利推广** 页面，免费注册成为代理商，获得绑定下级用户的资格。详细信息请查看 <a href="/rebate/rebate" target="_blank">代理商返利文档</a>。

2. **绑定下级用户**

   将您的返利码或绑定链接发送给需要接收优惠券的用户，用户绑定后即成为您的下级用户。

   :::tip[额外收益]
   用户绑定成为下级后，不仅可以接收您赠送的优惠券，您还可以从他们的消费中获得返利收益。详细信息请查看 <a href="/rebate/rebate" target="_blank">代理商返利文档</a>。
   :::

3. **赠送优惠券**

   在 **财务中心 - 我的卡包** 中，找到可赠送的优惠券，点击 **赠送** 按钮。

   ![赠送优惠券](@assets/finance/coupon/1.webp)

4. **输入用户信息**

   在弹出的模态框中输入下级用户的 **用户 ID**（可以向下级用户询问），点击 **确认赠送** 即可。

5. **赠送成功**

   赠送成功后，优惠券将从您的卡包转移到下级用户的卡包中，下级用户可以立即使用。

</Steps>

:::caution[赠送注意事项]

- 赠送前请确认下级用户的用户 ID，赠送后无法撤销
- 已赠送的优惠券从您的卡包中移除
- 赠送的优惠券有效期不会延长
- 只能赠送给您的直属下级用户

:::

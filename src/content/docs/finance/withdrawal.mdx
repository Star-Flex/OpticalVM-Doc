账户余额提现功能允许您将钱包中的余额转出至您的支付账户。余额来源包括退款金额、充值余额等，提现时可选择**订单原路退回**或**支付宝提现**两种方式。

:::tip[温馨提示]

- 余额也可直接用于购买服务，无需提现
- 订单原路退回需在订单额度范围内，支付宝提现最低 ¥1.00
- 支付宝提现前请确认账户已完成实名认证
- 如遇到账时间异常，可通过工单或在线客服反馈
- 提现不收取任何手续费

:::

## 提现流程

<Steps>

1. **发起提现申请**

   前往财务中心 - 账户钱包，点击 **提现** 按钮发起提现申请。

   ![提现按钮](@assets/finance/withdrawal/1.webp)

2. **管理员审核**

   提交提现申请后，系统将自动通知管理员进行审核。审核时间通常为 1-3 个工作日。

3. **提现到账**

   审核通过后，提现金额将按照您选择的方式退回至对应账户。

   - **订单原路退回**：24 小时内到账
   - **支付宝提现**：1-3 个工作日到账（不含周六周日及节假日）

4. **未到账处理**

   如果超过预计到账时间仍未收到款项，请通过以下方式联系我们：

   - **工单系统**：提交工单说明情况
   - **在线客服**：联系在线客服获取帮助

</Steps>

## 提现方式说明

<Tabs>
<TabItem label="订单原路退回">

:::tip[提示]

**适用场景：** 退款金额提现、希望快速到账（24 小时内）、需要退回至原支付方式

- 提现金额受**两个限制**：订单可提现额度 和 账户余额，取较小值
- 不同支付方式的订单需分别提现
- 订单额度用完后，该订单无法继续提现

:::

#### 什么是订单原路退回

订单原路退回是指将账户余额按照原支付订单的支付方式（微信、支付宝等）退回至原支付账户。这种方式基于**订单池**机制，每个支付订单都有对应的可提现额度。

#### 订单池机制

系统会记录您所有已支付的订单，形成一个"订单池"。每个订单都有：

- **订单实付金额**：购买时实际支付的金额
- **已提现金额**：从该订单已经提现的累计金额
- **可提现额度**：剩余可提现金额

**核心公式：**

```txt
单个订单可提现金额 = 订单实付金额 - 该订单已提现金额
```

#### 计算示例

假设您的账户状态如下：

- **账户余额**：¥10.00
- **订单池**：
  - 订单 A（微信支付）：实付 ¥100.00，已提现 ¥0，可提现 ¥100.00
  - 订单 B（支付宝）：实付 ¥5.00，已提现 ¥0，可提现 ¥5.00

**提现操作：**

```
操作 1：选择订单 B (支付宝)，提现 ¥4.00
  - 提现后订单 B 状态：实付 ¥5.00，已提现 ¥4.00，可提现 ¥1.00
  - 账户余额：¥10.00 - ¥4.00 = ¥6.00

操作 2：再次选择订单 B (支付宝)，提现 ¥1.00
  - 提现后订单 B 状态：实付 ¥5.00，已提现 ¥5.00，可提现 ¥0
  - 账户余额：¥6.00 - ¥1.00 = ¥5.00

此时订单 B 额度已用完，无法继续从订单 B 提现。
但可以选择订单 A (微信) 提现，最多可提现 ¥5.00（受账户余额限制）。
```

</TabItem>

<TabItem label="支付宝提现">

:::tip[提示]

**前置条件：** 账户已完成实名认证、提现支付宝必须为账户本人所有、首次提现可能需要绑定支付宝账户

**适用场景：** 充值余额提现、无原路退回订单的余额、订单池额度已用完的余额

:::

#### 什么是支付宝提现

支付宝提现是指将账户余额直接提现至您的支付宝账户，不受订单池限制。

#### 提现规则

- **最低提现金额**：¥1.00
- **最高提现金额**：账户可用余额
- **到账时间**：1-3 个工作日（不含周六周日及节假日）

</TabItem>
</Tabs>

:::caution[重要提示]

- 提现申请提交后若需撤销，请通过**工单系统联系客服**处理
- 支付宝提现必须使用**账户本人的支付宝**，否则审核不通过
- 提现**不收取任何手续费**，实际到账金额等于提现金额
- 账户余额不足时，无法发起提现申请

:::

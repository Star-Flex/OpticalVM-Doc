个人实名认证适用于个人用户，通过人脸识别技术验证您的真实身份。完成个人实名认证后，您可以购买和使用原光云的个人类云产品。

## 准备工作

在开始实名认证之前，请确保您已做好以下准备：
:::danger[认证前置条件]
**个人实名认证采用扫脸认证方式，需要您的人脸信息已录入公安部数据库。**

如果您从未办理过二代身份证或从未在公安机关采集过人脸照片，将无法通过扫脸认证。建议您前往当地公安机关更新身份证信息后再进行实名认证。
:::

**准备清单：**

1. ✅ 满足上方认证前置条件
2. ✅ 准备好摄像功能正常的手机设备（移动端认证需要）
3. ✅ 准备好本人有效身份证件

## 认证流程

根据您使用的设备类型，选择对应的认证方式：

<Tabs>
<TabItem label="PC 电脑端认证" icon="laptop">

<Steps>

1. **访问实名认证页面**

   在电脑浏览器中打开原光云实名认证页面，输入真实姓名和身份证号码，点击 **提交核验**。

   ![user-scan](@assets/user/auth/1.webp)

2. **扫描二维码完成认证**

   生成二维码后，使用手机设备打开微信、支付宝、QQ 等已授权相机权限的应用扫描网页生成的二维码，按照提示完成人脸识别认证。

   ![user-scan-2](@assets/user/auth/2.webp)

</Steps>

</TabItem>

<TabItem label="手机/移动端认证" icon="phone">

移动端设备可以直接访问实名认证页面，输入姓名和身份证号码。点击 **提交核验** 后可以直接跳转到人脸认证页面进行扫脸验证。

:::caution[注意事项]
使用移动端认证时，请务必允许浏览器访问摄像头权限，否则无法完成人脸识别。
:::

</TabItem>
</Tabs>

## 提升认证通过率的建议

为了确保人脸核验顺利通过，请注意以下事项：

### 环境要求

- **光线充足**：选择光线明亮的环境，避免背光或过暗
- **面部清晰**：摘下眼镜、帽子、口罩等遮挡物
- **位置适当**：调整位置确保人脸完整出现在识别框内，无遮挡

### 认证失败常见原因

如果多次认证不通过，可能是以下原因：

**1. 照片质量问题**

- 活体照片质量较低或与身份证照片差异过大
- **解决方法**：更换较好光照环境，调整位置保证人脸无遮挡后重新认证

**2. 设备安全检测失败**

为了保证认证的安全性，系统会对认证设备、身份、人脸等环境进行安全检测，若检测到可能存在风险会判定认证结果不通过。

**常见风险原因：**

- 设备检测到疑似存在劫持摄像头、被 Root、注入、模拟器等高风险特征
- 设备或身份存在一定时间内高频重复认证行为

**排查处理方法：**

- 卸载设备上可能安装的各种多开、分身、虚拟环境等软件或插件
- 恢复设备系统初始安全环境后重试
- 避免短时间内频繁尝试认证

:::tip[认证技巧]

- 确保本人操作，不要使用他人照片或视频
- 保持面部表情自然，按照提示完成指定动作
- 若为同一人且多次失败，可稍后再次尝试或联系客服

:::

## 常见问题

### 认证失败怎么办？

如果认证失败，请先查看失败原因提示，然后根据上述建议调整环境和设备后重新尝试。如果持续失败，请联系客服或提交工单寻求帮助。

### 认证需要多长时间？

扫脸认证通常在 1-3 分钟内即可完成，审核结果会实时反馈。

> 按照中国国家法律（如《反电信网络诈骗法》和《网络安全法》）要求，购买原光云产品或服务前，必须先完成实名认证。没有实名认证，您将无法购买、续费或使用大部分云服务。

原光云账号的实名认证包括个人实名认证和企业实名认证。在购买原光云产品或服务之前，您必须完成实名认证。

:::tip
**个人实名认证**适合个人用户，可使用的云产品有限。认证操作步骤详见个人实名认证。

**企业实名认证**适合公司或组织，可使用的云产品更全。认证操作步骤详见企业实名认证。

:::

### 认证类型对比

| 类型             | 个人实名认证                       | 企业实名认证                                   |
| ---------------- | ---------------------------------- | ---------------------------------------------- |
| **适用对象**     | 个人用户                           | 企业、个体工商户、政府、事业单位、学校、组织等 |
| **认证方式**     | 个人扫脸认证                       | 暂不开放，请联系客户经理处理                   |
| **发票信息**     | 开具个人，企业抬头的增值税普通发票 | 开具企业抬头的增值税专用发票或增值税普通发票   |
| **可使用云产品** | 仅能使用个人类型的云产品           | 可用个人和企业类云产品                         |
| **认证数量限制** | 一个身份证最多认证 1 个账号        | 一个企业主体最多认证 10 个账号                 |

:::tip[温馨提示]
如果企业账号的认证数量无法满足您的业务诉求，请联系您的客户经理进行处理。
:::

## 实名认证的限制

### 认证规则

| 规则项           | 具体要求                                                                             |
| ---------------- | ------------------------------------------------------------------------------------ |
| **实名认证类型** | 一个原光云账号只能认证一个主体，即一个账号只能对应一个个人实名认证或一个企业实名认证 |
| **人脸信息要求** | 个人扫脸认证需要身份证人脸信息已录入公安部数据库，未采集过人脸照片的用户无法通过认证 |
| **认证解绑**     | 实名认证后不可解绑，无需账号请直接注销账号                                           |

:::caution[重要提示]

- 实名认证一旦完成，无法更改或解绑
- 请确保填写的信息准确无误

:::
